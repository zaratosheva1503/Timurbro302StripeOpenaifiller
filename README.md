# 🚀 Zarif Card Automation

Chrome/Edge extension to automatically fill payment checkout forms with generated cards for OpenAI and Stripe pages.

**Version:** 5.0.0

## ✨ Features

- 🚀 Auto-generates valid cards using API
- 🎯 Works on **OpenAI Payment Pages** (pay.openai.com)
- 🎯 Works on **Stripe Checkout Pages** (checkout.stripe.com)
- 🇰🇷 Pre-configured with South Korean billing address
- 💳 **Default BIN:** `625814260209xxxx` (Expiry: 04/2027)
- 👤 Random Korean name generation
- ⚡ One-click auto-fill functionality

## 📦 Installation

### Google Chrome

1. **Download this repository**
   ```bash
   git clone https://github.com/yourusername/OpenAi-stripe-payment-autofill-Zarif-automation-.git
   ```
   Or download ZIP and extract it

2. **Open Chrome Extensions**
   - Go to `chrome://extensions/`
   - Enable **"Developer mode"** (toggle in top right corner)

3. **Load the Extension**
   - Click **"Load unpacked"**
   - Select the folder containing the extension files
   - Extension icon will appear in your toolbar! 🎉

### Microsoft Edge

1. **Download this repository**
   ```bash
   git clone https://github.com/yourusername/OpenAi-stripe-payment-autofill-Zarif-automation-.git
   ```
   Or download ZIP and extract it

2. **Open Edge Extensions**
   - Go to `edge://extensions/`
   - Enable **"Developer mode"** (toggle in bottom left corner)

3. **Load the Extension**
   - Click **"Load unpacked"**
   - Select the folder containing the extension files
   - Extension icon will appear in your toolbar! 🎉

## 🎯 How to Use

### For OpenAI Payment Page

1. **Connect to a VPN with South Korea server** 🇰🇷
   - Use any VPN that supports South Korea: **NordVPN**, **ProtonVPN**, **CyberGhost**, **Urban VPN**, etc.
2. **Go to OpenAI Payment Page** (`pay.openai.com`)
3. **Click the extension icon** in your toolbar
4. **Click "Generate & Auto Fill"**
5. **Done!** Card and billing details are filled automatically

### For Stripe Checkout Page

1. **Connect VPN to South Korea** 🇰🇷
2. **Open any Stripe checkout page** (`checkout.stripe.com`)
3. **Click the extension icon**
4. **Click "Generate & Auto Fill"**
5. **Done!** All fields are filled automatically

## 📋 Pre-Cards Feature

You can also generate cards in advance:

1. Go to **"Pre-Cards"** tab in the extension
2. Click **"Generate 10 Pre-Cards"**
3. Cards are saved locally
4. When on a payment page, click **"Use Now"** next to any card

## ❓ Troubleshooting

**Extension not appearing in toolbar?**
- Make sure Developer Mode is enabled
- Click the puzzle icon and pin the extension

**Auto-fill not working?**
- Refresh the payment page
- Make sure VPN is connected to South Korea
- Try clicking "Generate & Auto Fill" again

**Cards not generating?**
- Check your internet connection
- The API may be temporarily unavailable
